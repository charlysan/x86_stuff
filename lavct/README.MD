# lavct

Logic Analyzer Video Capture Tool

## Installation and running

```
pip install pygame
```

```
python lavct.py <CSV-FILE>
```

## Usage

- Change pixel clock period: `a` and `d` keys
- Change back porch value: `q` and `e` keys.
- Change colors:
   * `8` -> amber
   * `9` -> green
   * `0` -> white
